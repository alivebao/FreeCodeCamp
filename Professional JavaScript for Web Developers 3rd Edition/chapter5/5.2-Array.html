1. There are 3 methods to create Array: new Array/Array/direct init with a "[]"
2. Length of Array can rewrite after Array created.
If lenth reseted bigger than original, it will filled by "undefined".
If lenth restedd smaller than original, the more out of part will deleted. 3. Base 2, we can add length and content of Array dynamic 
Test Code:
//Ex1
var arr1 = new Array(5);
var arr2 = Array(3);
var arr3 = [1, "df", null, "ac"];   //array in js can make up with different types
console.log(arr1);
console.log(arr2);
console.log(arr3);
//Ex2
arr3.length = 50;
console.log(arr1);
console.log(arr2);
console.log(arr3);
//Ex3
var aimLength = arr3.length + 10;
/*
WARN: we can't just use (arr3.length + 10) in for(..) like 
for(var i = arr3.length; i < arr3.length + 10 ; ++i){...},
cause length of Arr is dynamic change, it will never stop 
*/
for(var i = arr3.length; i < aimLength ; ++i)
    arr3[i] = i;
